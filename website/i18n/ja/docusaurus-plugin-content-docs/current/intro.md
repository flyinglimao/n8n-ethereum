# はじめに

**n8n-nodes-ethereum** ドキュメントへようこそ！このコミュニティノードパッケージは、viemを使用した統一されたノード構造で、n8nに包括的なEthereumブロックチェーン統合を提供します。

## 機能

- ✨ **統一されたノード構造**：リソース/オペレーションパターンを使用した単一のEthereumノードで、よりクリーンなUXを実現
- 🔐 **安全な認証情報**：読み取り操作用のオプションのウォレットを備えた、個別のRPCおよびアカウント認証情報
- 🔄 **10種類のリソースタイプ**：Account、Block、Transaction、Contract、ERC20、ERC721、ERC1155、ENS、Gas、Utils
- ⚡ **トリガーサポート**：新しいブロック、コントラクトイベント、トランザクションをリアルタイムで監視
- 🌐 **マルチチェーン**：Ethereum、Polygon、BSC、Arbitrum、Optimism、Avalanche、Fantom、Base、カスタムネットワークをサポート
- 🪙 **完全なトークン規格**：ERC20、ERC721、ERC1155で、ABIを自動処理
- 📦 **viemで構築**：Ethereumインタラクション用のモダンで型安全なTypeScriptライブラリ

## 含まれるもの

このパッケージには2つの強力なノードが含まれています：

- **Ethereumノード**：ブロックチェーン操作を実行するための通常のノード
- **Ethereum Triggerノード**：ブロックチェーンイベントを監視するためのトリガーノード

## はじめに

n8n-nodes-ethereumを使い始めるには：

1. n8nインスタンスに[パッケージをインストール](./installation.md)
2. RPCとアカウントの[認証情報を設定](./credentials.md)
3. ブロックチェーンと対話するための[利用可能なリソース](./resources/overview.md)を探索

## ユースケース

- スマートコントラクトイベントを監視して通知を送信
- 自動化されたトークン転送を実行
- スマートコントラクトデータの読み取りと書き込み
- ブロックチェーントランザクションをリアルタイムで追跡
- ENSドメイン解決を統合
- DeFi自動化ワークフローを構築
- NFT操作を管理
